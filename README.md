# SCI-Bot - AI-Powered Science Education App

## Overview
Flutter mobile application for Grade 9 Science students in the Philippines.

## Tech Stack
- **Framework**: Flutter 3.x
- **State Management**: Riverpod
- **Local Storage**: Hive
- **Navigation**: GoRouter
- **API Integration**: Dio + Retrofit (OpenAI GPT)

## Development Status
- [x] Phase 1, Day 1: Project initialization complete
- [ ] Phase 1, Day 2: Theme system (upcoming)

## Setup Instructions
1. Install Flutter SDK (3.0+)
2. Clone repository
3. Run `flutter pub get`
4. Download fonts to `assets/fonts/`
5. Run `flutter run`
